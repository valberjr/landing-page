# retro snake landing

minimalist retro arcade-inspired personal landing page built with next.js.

## features

- retro arcade-inspired interface
- fully playable snake game
- dark/light mode support
- responsive layout
- keyboard + mobile controls
- local high score persistence
- minimalist pixel aesthetic

## stack

- next.js
- react
- typescript
- tailwind css
- next-themes
- vercel analytics

## development

```bash
npm install
npm run dev
```

open:

```text
http://localhost:3000
```

## build

```bash
npm run build
```

## deploy

optimized for vercel deployment.
